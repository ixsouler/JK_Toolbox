# JK工具箱 V1.0

## 简介

JK工具箱是一个AutoCAD辅助工具集成平台，采用模块化设计，统一管理多个实用工具。基于AutoLISP开发，支持DCL对话框交互，并预留了插件扩展接口，方便根据需求添加新功能。

------

## 功能列表

| 工具 | 命令 | 功能说明 |
| :--- | :--- | :--- |
| **批量查重** | `JK_FindDup` | 查找选中范围内的重复文字内容，自动按组分配不同颜色进行标记 |
| **批量替换** | `JK_Replace` | 支持最多3组文本同时替换，可记忆上次使用的替换对，方便重复操作 |
| **文字套椭圆** | `JK_TextEllipse` | 为选中的文字中心添加固定大小的椭圆（长轴半径5，短轴半径2.5），椭圆特性随当前图层 |

------

## 安装方法

### 方法一：加载LSP文件

1. 在AutoCAD中执行 `AP` 或 `APPLOAD` 命令
2. 选择 `JK_Toolbox.LSP` 文件
3. 点击"加载"

### 方法二：添加到启动组（自动加载）

1. 执行 `AP` 命令
2. 点击"启动组"区域的"内容"按钮
3. 添加 `JK_Toolbox.LSP` 文件
4. 重启AutoCAD后自动加载

## 使用方法

### 主命令

| 命令 | 功能 |
| :--- | :--- |
| `JK` | 打开工具箱主界面，通过可视化按钮选择工具 |
| `JKPLUGINS` | 查看所有已注册插件的加载状态 |
| `JKHELP` | 显示帮助信息及命令列表 |

### 工具使用说明

**批量查重（JK_FindDup）**

1. 运行命令后，框选需要检查的单行文字（TEXT）
2. 程序自动识别重复内容并分组显示
3. 确认后自动为每组分配不同颜色（颜色自动轮换，避免与原始颜色冲突）
4. 支持取消操作

**批量替换（JK_Replace）**

1. 运行命令后弹出替换设置对话框
2. 最多可设置3组"替换前/替换后"文本对
3. 选择替换范围（全部文本 / 手动选择）
4. 自动执行替换并输出统计结果
5. 替换对会自动保存，下次打开对话框时沿用

**文字套椭圆（JK_TextEllipse）**

1. 运行命令后，选择需要添加椭圆的文字对象（支持单行文字TEXT和多行文字MTEXT）
2. 程序自动计算每个文字的中心点
3. 在中心位置绘制固定大小的椭圆（长轴半径5，短轴半径2.5）
4. 椭圆颜色、线型、线宽完全跟随当前图层（ByLayer）
5. 支持预先选中对象后执行命令，提高操作效率

## 扩展插件

### 添加新插件步骤

1. **注册插件**：编辑 `*JK-plugin-list*` 列表，按格式添加条目

   ```lisp
   ("命令名" "显示名称" "功能描述" T)
   ```

   示例：

   ```lisp
   ("JK_MyTool" "我的工具" "自定义功能描述" T)
   ```

2. **定义命令函数**：创建对应的 `c:命令名` 函数（建议以 `JK_` 为前缀）

3. **添加对话框按钮**：在 `JK:create_main_dialog` 函数中添加对应的按钮控件

4. **绑定按钮动作**：在 `c:JK` 和 `JK:setup_dialog` 中添加按钮响应逻辑

### 插件命名规范

- 统一使用 `JK_` 前缀（如 `JK_FindDup`、`JK_Replace`、`JK_TextEllipse`）
- 避免与AutoCAD内置命令或其他插件冲突

## 编译方法

### 生成VLX文件（保护源代码）

1. 在AutoCAD中执行 `VLISP` 命令打开Visual LISP IDE
2. 点击菜单：文件 → 生成应用程序 → 新建应用程序向导
3. 选择"专家"模式
4. 添加 `JK_Toolbox.LSP` 作为源文件
5. 指定输出目录（如 `release\`）
6. 生成 `JK_Toolbox.VLX` 文件
7. 分发时仅提供VLX文件即可
